Pengenalan HTML, CSS, JavaScript, PHP, Node.js, dan Ruby 

(HyperText Markup Language) adalah bahasa markup untuk membuat struktur halaman web. HTML menggunakan tag-tag khusus untuk mendefinisikan elemen-elemen web.
            




            


            
            
CSS(Casding Style Sheets)
 digunakan untuk mendesain dan mempercantik tampilan halaman web. Dengan CSS, anda dapat mengatur warna font, layout, dan banyak lagi.

            




            


            
            
 JavaScript  memberikan interaktivitas pada halam web. Misalnya, anda dapat membuat animasi, Validasi formulir, atau fitur dinamis lainnya menggunakan JavaScript. 
            




            


            
            
 PHP(Hypertext Preprocessor)
 adalah bahasa pemograman server-side yang digunakan untuk mengembangkan aplikasi web dinamis. PHP sering digunakan bersama dengan database seperti MySQL untuk menyimpan dan mengambil data.

            




            


            
            
 Node.js  adalah platform runtime JavaScript yang berjalan di server. Node.js memungkinkan pengembang untuk membuat aplikasi backend menggunakan JavaScript, sehingga memudahkan integrasi antara frontend dan backend. 
            




            


            
            
 Ruby
 adalah bahasa pemrograman yang sederhana dan fleksibel, sering digunakan bersama framework Rails (Ruby on Rails) untuk pengembangan web. Ruby dikenal karena sintaksnya yang mudah dipahai dan produktivitasnya yang tinggi.
